### 1. KlonД“t repozitoriju


### 4. DatubДҒze un sДҒkotnД“jie dati


PД“c seeding izveidojas ЕЎie konti (parole visiem: `password`):

### 6. Palaist serveri


Atver [http://localhost:8000](http://localhost:8000)
